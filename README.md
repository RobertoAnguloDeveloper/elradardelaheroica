📡 El Radar de La Heroica - Landing Page

Esta es la landing page oficial para El Radar de La Heroica, una plataforma digital dedicada a la cobertura de sucesos, movilidad y denuncia ciudadana en tiempo real en la ciudad de Cartagena de Indias.

El objetivo de esta página es servir como embudo principal para captar reportes anónimos de la ciudadanía y redirigir el tráfico hacia las redes sociales oficiales del medio (Facebook, Instagram, WhatsApp).

🚀 Tecnologías Utilizadas

Este proyecto fue desarrollado priorizando la velocidad de carga y la adaptabilidad a dispositivos móviles (Mobile-First):

HTML5: Estructura semántica.

Tailwind CSS (v3 vía CDN): Framework de utilidades para un diseño rápido, moderno y altamente responsivo.

FontAwesome (v6.4): Implementación de iconografía vectorial ligera.

✨ Características Principales

Diseño de Alto Impacto: Paleta de colores basada en alertas (Rojo/Oscuro) para transmitir urgencia e inmediatez periodística.

Ticker de Última Hora: Animación CSS pura que simula un cintillo de noticias en constante movimiento.

Sección de Denuncia (Call to Action): Interfaz optimizada para generar confianza y fomentar el envío de material multimedia de forma 100% anónima.

Totalmente Responsivo: Adaptado perfectamente para visualización en smartphones, tablets y pantallas de escritorio.

🛠️ Uso y Despliegue

Al estar construida con tecnologías frontend estáticas (HTML/CSS), esta landing page no requiere de un servidor backend complejo para funcionar en su etapa inicial.

Clona este repositorio:

git clone [https://github.com/RobertoAnguloDeveloper/elradardelaheroica.git](https://github.com/RobertoAnguloDeveloper/elradardelaheroica.git)


Abre el archivo index.html en cualquier navegador web moderno para previsualizar.

Despliegue sugerido: Puede ser alojada de forma gratuita y en segundos utilizando servicios como GitHub Pages, Vercel, o Netlify.
